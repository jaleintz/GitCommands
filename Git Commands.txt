Git Commands

Global settings
- git config --global user.name "John Leintz"
- git config --global user.email "jaleintz@gmail.com"
- git config --global init.default branch main
- git config --list

Initialize
- git init
- git status

Track Status for one
- git add index.htm
- git rm --cached docA.txt

Git Ignore
- .gitignore

Track Status for all
- git add --all or git add -A or git add .

Commit
- get commit -m "First commit"

Diff
- get diff

Work File - Staging - Commit

Commit and bypass stating
- git commit -a -m "Comments"

Remove a file
- git rm filename.txt

Restore a file
- git restore --staged filename.txt

Rename
- git mv "docB.txt" "docC.txt"

Log
- git log
- git log --oneline
- git log -p

Amend
git commit -m "Updated description" --amend

Branch
- git branch
- git branch ChangeDoc
- git switch ChangeDoc
- got switch -c NewBranch

Delete Branch
- git branch -d ChangeDoc

Merge
- git merge -m "merge back to master" ChangeDoc

Delete a branch
- git branch -D ChangeDoc

Push existing local rep to GitHub
- create a repo on GitHub
- git remote add origin
